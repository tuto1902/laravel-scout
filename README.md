

## Installation

````
git clone git@github.com:tuto1902/laravel-scout.git
````
````
cd laravel-scout
````
````
composer install
````
````
cp .env.example .env
````
````
php artisan key:generate
````
````
touch database/database.sqlite
````
````
php artisan migrate --seed
````
````
npm install && npm run prod
````
````
php artisan serve
````
